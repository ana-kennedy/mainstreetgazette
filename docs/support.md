# Main Street Gazette Support

For support, questions, or issue reports, use the public GitHub repository for Main Street Gazette:

https://github.com/ana-kennedy/mainstreetgazette

When reporting an issue, include:

- The device or simulator you used
- The app version
- What you expected to happen
- What actually happened
- Any relevant screenshots or error messages

Main Street Gazette is an independent feed reader and podcast player. It is not affiliated with, endorsed by, or sponsored by Disney or the listed publishers.
