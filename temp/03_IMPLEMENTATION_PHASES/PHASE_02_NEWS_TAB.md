# Phase 02 — News: Today's Gazette

## Goal
Replace the current picker-heavy news experience with a calm daily edition.

## Structure
- Arrival greeting, optional and concise.
- Page title: `Today's Gazette`.
- Optional date and edition summary.
- Single view button showing the current choice:
  - Today's Magic
  - My Magic
  - All Headlines
- Search the Gazette.
- Optional Editor's Note only when it adds value.
- Continuous headline list with adaptive story clusters.

## Actions
Card primary action: Read, Watch, Listen, or Explore depending on content.
Context/rotor actions: Save to Gazette Library, Share, Open Original, About This Source. Add to Collection only after save or from Library; do not force organization at save time.

## Remove
- Permanent picker/segmented control.
- Saved and unread choices from the News view menu.
- Technical filter and source controls.
- Empty sections.

## Exit criteria
- Current view is announced with position and state.
- One story card requires one swipe in simplified navigation.
- View changes preserve list position per view.
