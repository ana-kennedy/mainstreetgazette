# Phase 05 — For You, Gazette Library, and Collections

## Goal
Keep For You quiet and useful, while supporting users with hundreds of permanent saved references.

## Root sections, maximum three primary headings
- Continue Your Adventure
- The Gazette Library
- Today's Picks

Sections disappear when empty. Use proper headings for VoiceOver and braille navigation.

## Save model
A single Save action always adds the item to The Gazette Library. Never ask for a collection at initial save.

## Library smart shelves
- Enjoy Later
- Continue Reading
- Continue Watching
- Continue Listening
- Recently Added
- Ready Offline

## User collections
User-created, mixed-media collections such as Vacation 2027, Restaurants to Try, Accessibility Tips, or Halloween Ideas. Users create, rename, reorder, pin, move/copy items, and delete collections. Never create or reorganize a personal collection automatically.

## Suggestions
The app may gently offer a collection suggestion based on repeated saving, but only as an invitation and never as an automatic action.
